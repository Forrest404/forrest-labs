# NOUR — Network for Operational Updates & Response

Civilian safety reporting, verified and mapped in real time.

NOUR lets civilians in conflict and disaster zones report strikes, shelling, and other
hazards from any phone — no app, no account. An AI pipeline clusters the reports, scores
confidence, cross-references news and official sources, and publishes verified incidents to a
live public map. A separate NGO platform consumes that verified incident data for aid teams.

- Public map: `/map`
- Civilian report form: `/report`
- Health: `/health`, `/api/health`
